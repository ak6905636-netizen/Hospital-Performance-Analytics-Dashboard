# 🏥 MediInsight 360 – Hospital Analytics Dashboard

## 📌 Overview

**MediInsight 360** is an end-to-end **Power BI analytics solution** designed to transform raw hospital data into actionable insights.
The dashboard provides a **360-degree view of patient activity, financial performance, and operational efficiency**, enabling hospital stakeholders to make faster, data-driven decisions.

---

## 🎯 Problem Statement

Hospitals generate large volumes of data across patient care, billing, and operations, but lack a centralized system to analyze it effectively.
This results in:

* Limited visibility into **patient trends and department performance**
* Inefficient **resource allocation**
* Difficulty in tracking **revenue and operational KPIs**

---

## 💡 Solution

This project delivers an **interactive Power BI dashboard** that:

* Consolidates hospital data into a single analytical view
* Tracks key performance indicators (KPIs)
* Enables dynamic filtering and drill-down analysis
* Supports strategic and operational decision-making

---

## 📊 Dataset Information

* **Total Records:** ~200,000
* **Domain:** Healthcare
* **Key Fields:**

  * Patient_ID, Age, Gender, City
  * Department, Admission_Type
  * Admission_Date, Discharge_Date
  * Billing_Amount, Cost Components
  * Payment_Method, Insurance

---

## 🛠️ Tools & Technologies

* **Power BI Desktop** (Data Visualization)
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Calculated Measures & KPIs)

---

## 📈 Key Metrics (KPIs)

* 👥 **Total Patients:** 200K+
* 💰 **Total Revenue:** ~5 Billion
* 🏥 **Average Stay Duration:** ~15 Days
* 🔁 **Readmission Rate**
* 💵 **Revenue per Patient**

---

## 📊 Dashboard Features

* **Patient Insights**

  * Admission trends over time
  * Gender and city distribution

* **Financial Analysis**

  * Department-wise revenue
  * Payment method distribution
  * Cost breakdown

* **Operational Efficiency**

  * Average stay duration
  * Room type utilization
  * Department workload

* **Interactive Filters**

  * City, Department, Gender
  * Admission Type & Date
